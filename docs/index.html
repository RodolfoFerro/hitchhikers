About the club

Have you ever wondered what the internet (connection between computers), biology (e.g. spreading of a disease), and society have in common? There are relations and connections everywhere. All you need is an eye for them (and a bit of programming skills too!).

In this course, you will learn programming fundamentals, you will explore complex systems with network theory tools, and you will apply machine learning techniques to get insights about systems behavior.
